API Endpoints Reference

📊 Wealth & Financial Data

GET /api/wealth/summary
Get user's wealth summary including total assets, goals, and recent transactions.
Returns: wealth object, goals object, recent_30_days object
POST /api/ai/dashboard
Generate AI-powered financial dashboard with KPIs and insights.
Body: { date_from, date_to, prompt }
Returns: dashboard object with health_score, KPIs, insights, recommendations

💹 Market Data

GET /api/market/deposits
Get current deposit rates from Mexican banks.
Returns: rates array with bank, rate, term, type
GET /api/market/stocks?symbols=AMXL.MX,SPY,QQQ
Get stock market data from Yahoo Finance.
Query: symbols (comma-separated)
Returns: quoteResponse with price, change, currency
GET /api/market/crypto?ids=bitcoin,ethereum,solana
Get cryptocurrency market data from CoinGecko.
Query: ids (comma-separated crypto IDs)
Returns: price data with 24hr change and market cap

🤖 AI Chat & Support

POST /api/chat
Get financial advice from AI chatbot.
Body: { question: string }
Returns: { ok: boolean, answer: string }
GET /api/chat/history
Get chat conversation history (last 20 messages).
Returns: array of { question, answer, created_at }

ℹ️ System

GET /api/status
Check API and AI gateway status.
Returns: { ok, ai_configured, ai_model, version, features }
